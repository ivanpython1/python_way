# bash
```bash``` &ndash; стандартный язык оболочки Linux. С ней можно управлять файлами и ОС в целом.

## Движухи с файловой системой
### Перейти в папку
```commandline
cd path/to/dir
```
### Печать файлов в текущей папке
```commandline
ls
```

## Переместить файл
```commandline
mv file_name destination_dir_name
```
