# Extra problems

## 1. [Calculator](task_1.py)
**Examples**:
```commandline
Input a: 8
Input b: 4
a = 8
b = 4
a + b = 12
a - b = 4
a * b = 32
a / b = 2
```

## 2. [Names](task_2.py)
**Examples**:
```commandline
Input first_name: Ivan
Input second_name: Ivanovich
Input last_name: Petrov
FSL format: Ivan Ivanovich Petrov
LFS format: Petrov Ivan Ivanovich
```
> **__Note__**: Don't use commas in ```print```
